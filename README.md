## Mi prompt navegacion 3 pantallas
Lenguaje Dart flutter, Nivel principiante, ejemplo de navegación entre 3 paginas utilizando rutas nombradas, desde main llamar a pagina 1 , en la pagina 1 mostrar appbar (titulo: "inicio" color blanco con fondo naranja, y en body en una columna mostrar texto "Pagina inicial" más un botón para ir a la pagina 2) en la pagina 2 (mostrar el appbar "Segunda Pagina" color naranja con fondo blanco y en body mostrar una imagen desde la red más un botón para ir a la pagina 3), en la pagina 3 (en appbar mostrar titulo "Tercera Pagina" color negro con fondo naranja y en body mostrar un container de 200x200 naranja claro ). Colores gradientes. Debe ser atractivo, fácil navegación y mostrar un código en un solo archivo.
